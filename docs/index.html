Bienvenido a Custom Discord Bot

Un bot de Discord robusto, modular y altamente personalizable. Diseñado para ser el corazón de tu comunidad, integrando economía, moderación avanzada, diversión y capacidades de Inteligencia Artificial.

🛡️ Moderación Avanzada

Sistema de casos, advertencias automáticas (warns), timeouts, kicks y bans integrados con logs seguros.

💰 Economía Persistente

Trabajos, tienda de ítems, banco, recompensas diarias y una tabla de clasificación (leaderboard).

🤖 Integración IA

Soporte multi-proveedor (OpenAI, Mistral, Gemini, DeepSeek, etc.) para interactuar inteligentemente con los usuarios.

⚡ Comandos Híbridos

Todos los comandos responden tanto a comandos de barra (/) como a prefijo tradicional (!).

Soporte Completo para Slash Commands

El bot registra automáticamente los comandos de barra en tu servidor al iniciar, brindando una experiencia moderna y rápida.

Instalación y Configuración

Para levantar el bot en tu propio entorno, sigue estos pasos cuidadosamente.

1. Clonar e Instalar

git clone https://github.com/thedarker451-maker/Discord-Custom-Bot.git
cd Discord-Custom-Bot
npm install

2. Configurar Variables de Entorno

Renombra el archivo .env.example a .env y configura tus credenciales:

DISCORD_TOKEN=tu_token_aqui
PREFIX=!
NODE_ENV=production

# Configuración IA (Opcional)
PROVIDER=mistral
API_KEY=tu_api_key_aqui
¡Precaución!

Nunca compartas tu archivo .env ni lo subas a repositorios públicos. Contiene tus tokens y llaves privadas.

3. Iniciar el Bot

Puedes iniciar el bot en modo desarrollo o producción:

# Modo Producción
npm start

# Modo Desarrollo (se reinicia al guardar cambios)
npm run dev

⚠️ Sistema de Warns

El bot implementa un sistema progresivo de sanciones automáticas:

🟢 Warns 1-2 (Verde)

Solo registro. El usuario recibe notificación DM con el motivo.

🟡 Warn 3 (Amarillo)

Mute automático por 1 hora. Color: #DA9031

🟠 Warn 4 (Naranja)

Sanción severa registrada. Color: #EE5A34

🔴 Warn 5 (Rojo)

Ban automático permanente. Color: #FD2F00

Comandos de Gestión de Warns

/unwarn [caso_id] - Remueve una advertencia específica.
/clearwarns [usuario] - Limpia todas las advertencias.
/warns [usuario] - Muestra el conteo y detalles.

Sistema de Economía

El bot cuenta con un completo sistema económico (Custom Coins - NC) con subida de niveles, trabajos y banco.

💳 Billetera y Banco

Usa /balance para ver tu dinero. Guarda tu dinero seguro usando /depositar y sácalo con /retirar.

💼 Trabajos y Niveles

Usa /trabajar para ganar NC y Experiencia. A medida que subes de nivel, desbloqueas mejores profesiones (Doctor, CEO).

🏪 Tienda e Inventario

Mira la /tienda y usa /comprar para obtener picos o cañas. Revísalos en tu /inventario.

🎣 Minería y Pesca

Si tienes las herramientas, usa /minar o /pescaria para obtener recompensas aleatorias muy valiosas.

Otros Comandos de Economía

  • /daily - Reclama tu recompensa diaria. ¡Mantén la racha para bonos!
  • /transferir - Envía dinero a otro usuario.
  • /apostar - Apuesta tus monedas al 50/50.
  • /top - Mira el Leaderboard de los más ricos del servidor.

Integración con el Bot

oh-my-logo se ejecuta automáticamente al iniciar el bot con npm start o npm run dev. Si no está disponible, muestra un banner fallback con colores ANSI.

Uso como Módulo de Node.js

A partir de la versión 1.0.0, Custom Discord Bot se puede importar programáticamente en otros proyectos Node.js.

Requisito

Tu proyecto debe usar ESM (ECMAScript Modules). Asegúrate de tener "type": "module" en tu package.json.

Importación Básica

Puedes importar y arrancar el bot pasándole un token directamente o dejando que lea el .env.

import { startBot, client, db } from 'custom-discord-bot';

// Iniciar el bot programáticamente
await startBot('TU_TOKEN_OPCIONAL_AQUI');

// Escuchar eventos extra si lo deseas
client.on('messageCreate', (msg) => {
    if(msg.content === 'ping custom') {
        msg.reply('Pong desde el módulo!');
    }
});

// Acceder a la base de datos interna
const topUsers = await db.getEconomyLeaderboard(5);
console.log(topUsers);

API Exportada

Export Descripción
startBot(token?) Inicializa los comandos, eventos y conecta el bot a Discord. Retorna la instancia de Client.
client La instancia activa de discord.js Client.
db El gestor de base de datos interno (para economía, niveles, casos de moderación, etc.).
allCommands Array que contiene las definiciones y lógica de todos los comandos registrados.
