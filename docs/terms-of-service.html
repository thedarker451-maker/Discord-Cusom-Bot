2. Descripción del Servicio

Discord Custom Bot es un bot de Discord de código abierto diseñado para ofrecer funcionalidades de moderación, economía, gestión de tickets, inteligencia artificial y personalización avanzada. El Bot se ejecuta en un servidor del Cliente y se conecta a la API de Discord.

Naturaleza del servicio: El Bot opera como un software de terceros que interactúa con la API de Discord. No es un servicio alojado ni gestionado centralmente por el desarrollador.

3. Inteligencia Artificial — Xiaomi MiMo

3.1 Proveedor de IA

El Bot integra un sistema de inteligencia artificial basado en el modelo Xiaomi MiMo, proporcionado por Xiaomi Corporation a través de su API compatible con el estándar OpenAI. Esta integración permite a los usuarios realizar consultas de lenguaje natural directamente desde Discord.

Modelo utilizado: xiaomi/mimo-v2-flash — Modelo de lenguaje de gran escala desarrollado por Xiaomi para asistencia conversacional.

3.2 Naturaleza del Servicio de IA

  • Servicio de terceros: Las consultas de IA son procesadas por los servidores de Xiaomi. El Bot actúa únicamente como intermediario entre el usuario de Discord y la API de Xiaomi.
  • Sin garantía de exactitud: Las respuestas generadas por la IA son probabilísticas y pueden contener errores. No constituyen asesoramiento profesional, legal, médico ni financiero.
  • Uso bajo responsabilidad: El Cliente es responsable del uso que sus miembros den al sistema de IA y de informarles de esta naturaleza.

3.3 Disponibilidad

El servicio de IA depende de la disponibilidad de la API de Xiaomi. El desarrollador del Bot no garantiza la disponibilidad continua del servicio, ni se hace responsable por interrupciones, límites de tasa o cambios en la API de terceros.

4. Tratamiento de Datos Personales

4.1 Datos recopilados

Dato Propósito Retención
ID de usuario de Discord Gestión de economía, roles, tickets Eliminado con el bot
Mensajes de Discord Comandos, historial de tickets Solo en hilos de tickets (hasta cierre)
Consultas de IA Generación de respuestas No se almacenan en el Bot
Datos de economía Sistema de monedas y niveles Almacenamiento local o Redis
Token de Discord Autenticación del bot Almacenado en servidor local

4.2 Envío de datos a terceros

Importante: Cuando un usuario utiliza el comando de IA, su mensaje es enviado a los servidores de Xiaomi Corporation para ser procesado. El Bot no almacena ni registra estas consultas, pero el proveedor de IA puede tener sus propias políticas de retención de datos.

El Cliente debe informar a sus miembros que:

  • Las consultas de IA son procesadas por un servicio externo (Xiaomi).
  • No deben introducir información personal sensible en las consultas de IA.
  • El contenido generado por IA no es verificado ni garantizado.

4.3 Derechos del usuario

De conformidad con las regulaciones aplicables de protección de datos (incluyendo RGPD, CCPA y legislación local), los usuarios tienen derecho a:

  • Acceso: Solicitar información sobre los datos que el Bot almacena sobre ellos.

  • Eliminación: Solicitar la eliminación de sus datos del sistema.
  • Portabilidad: Solicitar sus datos en formato estructurado.

4.4 Seguridad

  • El Bot almacena datos en un servidor bajo el control del Cliente, no en servidores del desarrollador.
  • Las credenciales (tokens, API keys) se almacenan en archivos locales con encriptación opcional.
  • El Cliente es responsable de la seguridad de su servidor y de la configuración del Bot.
